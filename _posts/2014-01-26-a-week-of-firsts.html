This was a week of some big firsts for me. I created my first Ruby Gem and also my first API.

The Ruby Gem is called LyricGenius and returns lyrics and song information based on a user query. The Gem is powered by Mechanize, a powerful web scraper/crawler that itself is powered by the famous Nokogiri Gem.

LyricGenius works by submitting the user query on RapGenius.com. It then selects the top result and scrapes the respective song page for the lyrics, title and artist.

The API is powered by the gem and returns a JSON object with the aforementioned attributes. I built the API using Sinatra, which is like a super-lightweight Rails. It is currently hosted on Heroku.

Programming these tools were great learning experiences. The code itself is quite simple, but it was a great exercise to get familiar with building a hosted API and Ruby Gem.

Now that I got my toes wet without getting hurt, I feel much more comfortable building more complex Gems and API’s to power my own and other people’s web applications.

I have some great ideas and I am excited to get started coding them up.
